## 首页真实渲染验收修复（2026-06-16 16:20 UTC+8）

### 背景
人工打开正式站点发现首页真实渲染存在上线问题：官方当前主力区域观感只剩 1 个模型、国内人民币榜卡片不足、最新模型发现被单一平台变体刷屏，活动价与优惠区域还会把厂商网页正文/导航文本直接堆到首页。此前 audit 命令通过不等于正式域名 HTML 渲染正确。

### 修复内容
- 首页官方当前主力改为直接读取 `official_current_models` catalog，不再用价格榜严格过滤来决定主力展示。
- 官方当前主力标题改为“官方当前主力模型”，允许无价格但明确显示“价格待确认”。
- 国内人民币价格榜首页取消过严的 `homepageStrict / requireOfficialCurrent / maxSourceAgeHours` 过滤，保留隐藏旧模型与 diversity 限制。
- 最新模型发现从更宽候选池选取，并限制同 provider / model_family 刷屏；`unknown` 不再作为首页重点标签裸露，显示为“待确认”。
- 活动价与优惠区域过滤疑似爬虫正文、导航文本和超长描述；无法生成干净摘要时不在首页展示。
- 新增真实 HTML 验收脚本：`npm run audit:homepage-render`。
